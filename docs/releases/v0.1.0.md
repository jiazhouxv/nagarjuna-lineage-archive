## Status / 状态

This is a draft release note for the first structured release of Nagarjuna Lineage Archive.

这是龙树菩萨思想谱系资料库第一个结构化版本的发布说明草案。

## Summary / 概要

This release establishes the repository as a metadata-first, source-linked, machine-readable digital humanities archive for Nagarjuna-related Buddhist lineages, texts, doctrinal terms, timelines, branches, and relation data.

本版本将仓库建立为一个以元数据为核心、以来源链接为基础、面向机器可读的佛教数字人文资料库，用于整理与龙树菩萨相关的祖师谱系、典籍、教义术语、时间线、地域分支和关系数据。

The first active module is Tiantai, with early Sanlun scaffold planning now started for future expansion.

第一个重点模块是天台宗，同时已经开始为后续三论宗扩展建立初步脚手架规划。

## Source and full-text policy / 来源与全文政策

This release is metadata-first.

本版本以元数据为核心。

The repository records:

- source links
- canon identifiers
- bibliography metadata
- contributor-written summaries
- relation data
- status and review metadata

本仓库记录：

- 来源链接
- 藏经编号
- 参考书目元数据
- 贡献者原创摘要
- 关系数据
- 状态和审查元数据

The repository does not redistribute restricted Buddhist full-text corpora or copyrighted modern translations.

本仓库不再分发受限制的佛典全文语料，也不收录受版权保护的现代译文全文。

## Known limitations / 已知限制

- Many records remain `draft`, `partial`, or `needs_review`.
- Source references are improved but not exhaustive.
- Some records use general reference sources pending stronger academic or primary-source verification.
- Sanlun is only at the initial scaffold stage.
- More release QA should be completed before publishing the final GitHub release.

- 多数记录仍处于 `draft`、`partial` 或 `needs_review` 状态。
- 来源引用已有改进，但仍不完整。
- 部分记录暂时使用一般参考来源，后续应补充更强学术来源或原典来源。
- 三论宗目前只是初始脚手架阶段。
- 正式发布 GitHub release 前仍需完成更多发布前检查。
