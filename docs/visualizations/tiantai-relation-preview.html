这个 demo 要表达什么

真正的前台不只是列出人物和书名，而是让用户看到：龙树思想如何经中观、译经、祖师诠释、典籍系统、教义范畴、地域传播，形成天台宗模块中的多层网络。后续可把 YAML / JSON 数据导出后替换这里的 sample data。
