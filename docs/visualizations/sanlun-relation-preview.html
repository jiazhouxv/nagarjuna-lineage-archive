Nagarjuna Lineage Archive 的三论关系可视化预览。当前页面仅用于演示交互与数据结构，不是正式关系记录。

sample_only 静态 HTML5 无 CDN 无外部 JS locale: auto

项目说明

本项目以 metadata-first、source-linked、machine-readable 为原则，整理龙树及三论相关人物、典籍、术语和关系线索。仓库不复制受限制全文，只记录元数据、来源链接、原创摘要和待审关系。

功能预览

此页面展示未来关系图功能的最小原型：语言切换、节点详情、关系详情、图例、安全边界说明，以及后续由已审查 JSON 数据驱动的可能路径。

  • 点击节点查看人物、典籍、宗派或概念信息。
  • 点击关系线查看关系类型、证据强度和审查状态。
  • 支持 URL、手动、部署层 IP locale、浏览器语言四级语言选择。
