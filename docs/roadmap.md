# Roadmap / 路线图

## Purpose / 目的

This roadmap defines the short-term and long-term development plan for Nagarjuna Lineage Archive.

It is designed to keep the project maintainable, reviewable, and source-aware.

本文档定义龙树菩萨思想谱系资料库的短期和长期建设路线。

本路线图的目标是让项目保持可维护、可审查、重视来源依据。

## Application Readiness / 申请准备度

The project may be considered ready for open-source maintainer benefit applications when it has:

- Public repository
- Complete bilingual README
- Project scope documentation
- Source policy documentation
- Roadmap documentation
- Data schema documentation
- At least 8 person records
- At least 6 work records
- At least 3 branch records
- At least 10 relation records
- At least 1 validation tool
- Issues tracking missing data
- Pull requests showing reviewable maintenance
- At least 1 release
- Around 10 to 14 days of visible maintenance activity

当项目具备以下条件时，可以考虑申请开源维护者相关福利：

- 公开仓库
- 完整中英文 README
- 项目范围说明文档
- 资料来源政策文档
- 路线图文档
- 数据结构说明文档
- 至少 8 条人物记录
- 至少 6 条典籍记录

- 至少 10 条关系记录
- 至少 1 个校验工具
- 使用 issues 跟踪缺失资料

- 大约 10 到 14 天可见维护痕迹

## Long-Term Vision / 长期愿景

The long-term goal is to build a structured, source-aware, machine-readable archive for Nagarjuna-related Buddhist lineages and doctrinal reception across East Asian Buddhist traditions.

长期目标是建立一个结构化、重视来源、机器可读的资料库，用于整理龙树菩萨相关思想在东亚佛教诸宗中的谱系叙述、教义接受和历史传播。

Future modules may include:

- Tiantai
- Sanlun
- Huayan
- Chan
- Pure Land
- Faxiang
- Vinaya
- Esoteric Buddhism

未来可逐步建设：

- 天台宗
- 三论宗
- 华严宗
- 禅宗
- 净土宗
- 法相宗
- 律宗
- 密教
